## 💻 Sobre o projeto

AgencyWeb é um projeto institucional desenvolvido para praticar e demonstrar conhecimentos em HTML5 e CSS3.

O desafio proposto para este projeto foi a criação de uma página institucional moderna e responsiva, aplicando boas práticas de desenvolvimento web. Neste projeto, foram utilizados elementos semânticos do HTML para estruturar o conteúdo de forma organizada e acessível, além de CSS para estilização e adaptação a diferentes dispositivos.
